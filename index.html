HTML

  • The head element contains information about the webpage.
  • The head includes info such as language,metadata such as charset, http-equiv, viewport and title shown in the tabs
  • i.e. (html lang="en"), (meta charset="UTF-8), (meta http-equiv="X-UA=Copmpatible" content="IE=edge"), (meta name="viewport" content="width=device-width, initial-scale=1.0")

  • The body element represents the visible content shown to the user.
  • The body includes codes that dictate formatting, types of inserts, links, etc
  • i.e. (h1)&(h2) for headings, (ul),(ol),(li) for unordered, ordered lists and list items, (img)=images with (src)=sources such as URL and/or locations, (alt) for image titles as well as width and height settings, (br) for empty space, (a) for anchors to links (href)to point to an URL (a href="URL")

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element
  • There are inline CSS, internal & external CSS style sheets. External is most commonly used
  • inline: quick & permanent but not flexible if you want to change it later
  • internal: CSS that affects that page via classes & IDs
  • external: CSS is kept in a separate file and referred on the page
